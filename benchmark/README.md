# Benchmark

This is a pretty simple benchmark. From root directory, run: `python3 benchmark/run.py [number_circles]`. Default number of circles is 1000.

Some (non-rigorous) results:

|   Number  |  Time (s) |
|-----------|-----------|
| 10        | 0.0014    |
| 100       | 0.0043    |
| 1,000     | 0.17      |
| 10,000    | 7.24      |
| 100,000   | 222       |
